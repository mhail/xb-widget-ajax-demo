== Description ==

Demo of using the Xb_Widget_Ajax class to create Ajax enabled widgets.

== Installation ==

1. Copy the xb-widget-ajax-demo folder to the wp-content/plugins folder.
2. Activate the "Xb Ajax Widget Demo" plug-in in the wp-admin plugin screen.
3. Add the Xb_Widget_Ajax_Demo widget to a side-bar on the Widgets wp-admin screen.
